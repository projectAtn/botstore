# External Rollout Gate Checklist (v1)

This checklist is **strict** and fail-closed for external rollout decisions.

## Decision Rule

- **GO** only when every required gate item passes.
- **NO_GO** if any required gate item is missing, stale, or failing.
- Machine-readable packet: `research/external-rollout-checklist.json` (generated by `scripts/external_gate_packet.py`).

## Required Gate Items

1. **Trust policy in production mode**
   - Source: `research/trust_policy_v1.json`
   - Requirement: `mode` is exactly `production`.

2. **Exact identity pins (no scope widening in trust identities)**
   - Source: `research/trust_policy_production_template.json`
   - Requirements:
     - No `<ORG>/<REPO>` placeholders.
     - No wildcard identity matching.
     - Workflow identities pinned to exact refs (e.g., `@refs/heads/main`).

3. **Attestation enforcement active**
   - Sources:
     - `research/trust_policy_production_template.json`
     - `research/cryptographic-negative-matrix-report.json` (or trust-chain negative fallback)
   - Requirements:
     - Required attestations are declared.
     - Missing-attestation negative case fails closed.

4. **No scope widening**
   - Sources:
     - `research/launch-scorecard-*.json`
     - `research/openclaw-conformance-result.json`
   - Requirements:
     - Typed action-map checks pass.
     - Auth-bypass checks pass.

5. **Conformance green**
   - Source: `research/openclaw-conformance-result.json`
   - Requirement: health + typed action-map + auth-bypass pass.

6. **Alerts green**
   - Source: `research/alert-test-report.json`
   - Requirement: `all_pass == true`.

7. **Heartbeat fresh**
   - Source: `research/heartbeat.json`
   - Requirement: freshness within external gate SLO (currently 2 hours in packet script).

8. **7-day canary completed**
   - Source: `research/canary-week-report.json`
   - Requirements:
     - `days_observed >= 7`
     - `go_nogo == GO`
     - `open_incidents == 0`

## Supporting Artifact

- `research/phase-gate-summary.json` should report `go_nogo: GO`.
