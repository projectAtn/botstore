# BotStore 90-Day Operator Scorecard

## North star
**WAIA** (Weekly Active Installed Agents)

## Targets by phase
- **Days 1–30**: WAIA 2k → 6k
- **Days 31–60**: WAIA 6k → 14k
- **Days 61–90**: WAIA 14k → 25k

## KPI tree + weekly targets

### Supply
- Verified listings/week: 25 → 80
- Median listing SLA: <48h → <12h
- First-submit eval pass: 50% → 75%

### Demand
- New buyer org activations/week: 15 → 60
- Install→first-success: 35% → 65%
- 4-week buyer retention: 45% → 70%

### Monetization
- Weekly GMV: $8k → $120k
- Paid conversion: 10% → 28%
- Failed payment rate: <3% → <1%

### Trust / quality
- High-severity incident rate: <1.5 → <0.5 per 10k runs
- Top-workflow success: 60% → 85%
- Time-to-disable harmful pack: <30m → <5m

## Experiment backlog (first 10)
1. Embedded install in 2 runtimes
2. CLI publisher scaffold onboarding
3. Verification badge A/B
4. Top-10 curated bundles/stacks
5. Weekly creator analytics digest
6. User stack-sharing referral loop
7. Publisher referral program
8. Partner verified-integration co-launches
9. Programmatic SEO for top 50 jobs
10. Enterprise private-catalog pilots

For each experiment track:
- Hypothesis
- Leading signal (week 1)
- Success threshold (week 4)
- Owner
- Kill/iterate decision

---

## Owners (bot board)
- Strategy Bot: KPI governance, experiment prioritization
- Growth Bot: channels, partnerships, referral mechanics
- Content Bot: awareness + conversion narrative
- CTO Bot: infrastructure, QA, policy enforcement
- Quality Bot: contract/risk gates, verification, delisting

## Weekly governance format
- Mon: KPI + risk review
- Wed: product/QA status
- Fri: GTM + community review
- End of week: publish wins/misses + next actions
